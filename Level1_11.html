JIS Z 9002:1956 $ p_0 = 1.0% $ $ p_1 = 30% $

  1. $ n $ $ c $
  2. $ p_1 = 30% $ $ p_0 = 1.0% $

Quiero realizar una inspeccin por muestreo del tipo de norma de conteo JIS Z 9002:1956. Con el objetivo de aceptar lo ms posible lotes con una tasa de defectuosos mxima de $ p_0 = 1.0% $ y de rechazar lo ms posible lotes con una tasa de defectuosos mnima del $ p_1 = 30% $, el esquema de inspeccin por muestro sera:

  1. Tamao de la muestra $ n $, nmero de aceptaciones $ c $.
  2. Probabilidad de que un lote con una tasa de defectuosos de $ p_1 = 30% $, pase la inspeccin, y la probabilidad de que un lote con una tasa de defectuosos de $ p_0 = 1.0% $ no pase la inspeccin.

from scipy.stats import binom

# JIS Z 9002:1956 Datos de la tabla de muestro
SAMPLING_TABLE = [
    (5, 0), (7, 0), (10,0), (15, 1), (15,2), (20, 1), (20,3), (25, 4), (30, 6)
]

# Condiciones establecidas
p0 = 0.01  # Lmite superior de la tasa de unidades no conformes
           #para los lotes que se desean aceptar
p1 = 0.30  # Lmites inferiro de la tasa de unidades no conformes
           # para los lotes que se desean rechazar
alpha = 0.05  # L(p0) >= 1 - alpha
beta = 0.17   # L(p1) <= beta

# Bsqueda de n y c que cumplan las condiciones
def find_sampling_plan_from_table(p0, p1, alpha, beta, table):
    for n, c in table:
        L_p0 = binom.cdf(c, n, p0)  # Probabilidad de aceptacin para p0
        L_p1 = binom.cdf(c, n, p1)  # Probabilidad de aceptacin para p1

        if L_p0 >= 1 - alpha and L_p1 <= beta:
            return n, c
    return None, None

# Ejecucin
n, c = find_sampling_plan_from_table(p0, p1, alpha, beta, SAMPLING_TABLE)

print(f"Tamao de muestra n: {n}")
print(f"Nmero de aceptaciones c: {c}")

def get_sampling_plan(aql, lot_size, inspection_level="II"):
    # Criterios de muestreo de aceptacin basados en JIS Z 9015-1 (ISO 2859-1)
    # Datos de la tabla de especificaciones
    jis_table = pd.DataFrame({
        "Code": ["J", "K", "L"],  # Cdiogo
        "Lot Size Range": [(501, 1200), (1201, 3200), (3201, 10000)],
        "Sample Size": [80, 125, 200],
        "AQL 0.40%": [(1, 2), (1, 2), (2, 3)],
    })

    # Obtener la letra de cdigo correspondiente al tamao del lote
    selected_row = jis_table[(jis_table["Lot Size Range"].apply(lambda x: x[0] <= lot_size <= x[1]))]
    if selected_row.empty:
        raise ValueError("El tamao del lote est fuera del rango de aplicacin")

    sample_size = selected_row["Sample Size"].values[0]
    ac, re = selected_row["AQL 0.40%"].values[0]
    code = selected_row["Code"].values[0]

    return sample_size, ac, re, code

# Configuracin de parmetros
aql = 0.40  # Valor de AQL

$ \mu $ $ \mu_0 $ $ \mu_1 $ $ \sigma $ $ \bar{x} $

En la inspeccin por muestreo con base en criterios de medicin, se considera un lote bueno aquel cuya media poblacinal $ \mu $ de la caractersitica x es al menos $ \mu_0 $ y un lote malo aquel cuya media poblacional es como mximo $ \mu_1 $. Se asume que un mayor valor de la media es preferible. Adems, se supone que x sigue una distribucin normal y que el valor de su desviacin estndar $ \sigma $ es conocido. Se desea disear un plan de muestreo en el cual la aceptacin o rechazo del lote se determine comparando la media muestral $ \bar{x} $ de una muestra de tamao $ n $ con un valor de referencia para la aceptacin.

1.

100.0 98.0 $ \sigma = 1.0 $ $ \alpha = 0.05 $ $ \beta = 0.10 $

  1. n

Se desea aceptar en la medida de lo posible los lotes cuya media sea al menos 100.0, mientras que los lotes con una media de 98.0 o menos deben ser rechazados en la medida de lo posible. Se asume que la desviacin del es $ \sigma = 1.0 $, con $ \alpha = 0.05 $ y $ \beta = 0.10 $.

Determinar lo siguiente:

  • El tamao de la muestra n
  • El valor lmite inferior para la aceptacin

import math
from scipy.stats import norm

# Configuraci[on de parmetros
mu_0 = 100.0  # Media del lote que se desea aceptar
mu_1 = 98.0   # Media del lote que se desea rechazar
sigma = 1.0   # Desviacin estndar
alpha = 0.05  # Probabilidad del riesgo del productor
beta = 0.10   # Probabilidad de riesto del productor

# Clculo de K y K
K_alpha = norm.ppf(1 - alpha)
K_beta = norm.ppf(1 - beta)

# Clculo del valor lmite inferior de aceptacin Xbar_L
n_row = ((K_alpha + K_beta) * sigma / (mu_0 - mu_1)) ** 2
Xbar_L = mu_0 - K_alpha * (sigma / math.sqrt(n_row))

# Clculo del tamao de la muestra n
n = math.ceil(n_row)  # El tamao de la muestra se redondea hacia arriba
                      # el entero ms cercano

#
print(f"Tamao de la muestra n: {n}")
print(f"Valor lmite inferior de aceptacin Xbar_L: {Xbar_L:.2f}")
