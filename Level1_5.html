5.6 Análisis del punto de equilibrio¶

ある金型製造業では、固定費が３億円で、変動費率が0.4である。損益分岐点いくらか。

また、利益を０．６億円にするには、売り上げをいくらにしなければならないか。

En una empresa de fabricación de moldes, los costo fijos son de 300 millones de yenes y la tasa de costos vaiables es de 0.4. ¿Quál es el punto de equilibrio?

Además, ¿cuánto debe ser la venta para obtener una ganancia de 60 millones de yenes?

In [ ]:
# Costos fijos, Tasa de costors variables, Beneficio objetivo
fixed_cost = 300  # Costos fijos
variable_cost_rate = 0.4  # Tasa de costos variables
target_profit = 60  # Beneficio objetivo

# Calcular las ventas en punto de equilibrio
break_even_sales = fixed_cost / (1 - variable_cost_rate)
print(f'Las ventas en punto de equilibrio: {break_even_sales:.2f} millones')

# Calcular las ventas necesarios para alcanzar el beneficio objetivo
required_sales_for_target_profit = (fixed_cost + target_profit) / (1 - variable_cost_rate)
print(f'Las ventas necesarias para obtener un beneficio de {target_profit:.2f} millones: {required_sales_for_target_profit:.2f} millones')

# Los nuevos costos fijos después de reducir 10 millones
new_fixed_cost = fixed_cost - 10

# Calcular la nueva tasa de costos variables para alcanzaar el beneficio objetivo
# Las ventas se mantendrán en el nivel del punto de equilibrio
new_break_even_sales = break_even_sales
# Cálculo de la nueva tasa de costos variables
new_variable_cost_rate = 1 - (new_fixed_cost + target_profit) / new_break_even_sales
print(f'La nueva tasa de costos variables para obtener un beneficios de {target_profit:.2f} millones después de recucir 10 millones en costo fijos: {new_variable_cost_rate:.2f}')

Las ventas en punto de equilibrio: 500.00 millones
Las ventas necesarias para obtener un beneficio de 60.00 millones: 600.00 millones
La nueva tasa de costos variables para obtener un beneficios de 60.00 millones después de recucir 10 millones en costo fijos: 0.30

  • Tasa de costos variables = Costos variables / Ingresos

  • Ingresos = Costos fijos / (1 - Tasa de costos variables)

  • Punto de equilibrio (ingresos) = Costos fijos / (1 - Tasa de costos variables)

  • Tasa de costos variables = 1 ' (Ganancia + Costos fijos) / Ingresos

5.9 VE(Value Engineering)¶

機能達成にかかっている現状コストと本来の各機能が果たすためにかけなければならないコスト、すなわち目標コストを比較し、各機能の改善余地を明確にするために、機能評価をおこなう。

各機能分野の価値指数、改善余地を比較し、優先的に攻める機能分野を選択する。

Se realiza una evaluación de las funciones para comparar el costo actual necesario para lograr las funcionalidades y el costo ideal que cada función debe tener para cumplir su propósito, es decir, el costo objetivo, con el fin de identificar las áreas de mejora de cada función.

Se comparan los índices de valor y el margen de mejora de cada área funcional, y se selecciona el área funcional que debe ser priorizada para la mejora.

5.10¶

X、Yの二つの製品を製造する工場がある。各製品１単位をつくるための原材料A,Bの使用量、原料A、Bの月当たりの使用量の上限が与えられている。また、製品Xは一単位の販売につき一万円、製品Yは２万円の利益が見込まれる。このとき、利益を最大にする製品ＸとＹの製造量、ｘ単位とｙ単位を求めよ。

Hay una fábrica que produce dos productos, X y Y. Se nos dan las cantidades de materiales A y B que se utilizan para producir una unidad de cada producto, así como los límites de uso mensual de los materiales A y B. Además, se espera que el beneficio por la venta de una unidad del producto X sea de 10,000 yenes y el beneficio por la venta de una unidad del producto Y sea de 20,000 yenes. En este caso, se debe determinar la cantidad de productos X (x unidades) e Y (y unidades) a producir para maximizar el beneficio.

from scipy.optimize import linprog

# Función objetivo (dado que se desea maximizar el beneficio, ya que
# scipy.optimize.linprog minimiza, se inverte el signo)
c = [-10, -20]  # -x - 2y

# Condiciones de restricción
A = [[3, 3],  # 3x + 3y <= 48
     [1, 6]]  # x + 6y <= 36
b = [48, 36]

# Restricciones deno negatividad de las variables
x0_bounds = (0, None)
x1_bounds = (0, None)

# Resolver el problema de programación lineal
res = linprog(c, A_ub=A, b_ub=b, bounds=[x0_bounds, x1_bounds], method='highs')

# Salida de la solución óptima
print('Optimal value:', -res.fun, 'with x:', res.x[0], 'and y:', res.x[1])

# Función para calcular la cantidad óptima de producción
x_optimal = res.x[0]
y_optimal = res.x[1]

# x unidades de producto X y y unidades de producto Y
def calculate_profit(x, y):
    return 10 * x + 20 * y

# Calcular el benificio con la cantidad óptima de producción
profit_optimal = calculate_profit(x_optimal, y_optimal)
print(f"Beneficio con la cantidad óptima de producción: z = 10 * {x_optimal:.2f} + 20 * {y_optimal:.2f} = {profit_optimal:.2f} 万円")
