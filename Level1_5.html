5.6 Anlisis del punto de equilibrio

0.4

En una empresa de fabricacin de moldes, los costo fijos son de 300 millones de yenes y la tasa de costos vaiables es de 0.4. Qul es el punto de equilibrio?

Adems, cunto debe ser la venta para obtener una ganancia de 60 millones de yenes?

In[]:
# Costos fijos, Tasa de costors variables, Beneficio objetivo
fixed_cost = 300  # Costos fijos
variable_cost_rate = 0.4  # Tasa de costos variables
target_profit = 60  # Beneficio objetivo

# Calcular las ventas en punto de equilibrio
break_even_sales = fixed_cost / (1 - variable_cost_rate)
print(f'Las ventas en punto de equilibrio: {break_even_sales:.2f} millones')

# Calcular las ventas necesarios para alcanzar el beneficio objetivo
required_sales_for_target_profit = (fixed_cost + target_profit) / (1 - variable_cost_rate)
print(f'Las ventas necesarias para obtener un beneficio de {target_profit:.2f} millones: {required_sales_for_target_profit:.2f} millones')

# Los nuevos costos fijos despus de reducir 10 millones
new_fixed_cost = fixed_cost - 10

# Calcular la nueva tasa de costos variables para alcanzaar el beneficio objetivo
# Las ventas se mantendrn en el nivel del punto de equilibrio
new_break_even_sales = break_even_sales
# Clculo de la nueva tasa de costos variables
new_variable_cost_rate = 1 - (new_fixed_cost + target_profit) / new_break_even_sales
print(f'La nueva tasa de costos variables para obtener un beneficios de {target_profit:.2f} millones despus de recucir 10 millones en costo fijos: {new_variable_cost_rate:.2f}')

Las ventas en punto de equilibrio: 500.00 millones
Las ventas necesarias para obtener un beneficio de 60.00 millones: 600.00 millones
La nueva tasa de costos variables para obtener un beneficios de 60.00 millones despus de recucir 10 millones en costo fijos: 0.30

  • Tasa de costos variables = Costos variables / Ingresos

  • Ingresos = Costos fijos / (1 - Tasa de costos variables)

  • Punto de equilibrio (ingresos) = Costos fijos / (1 - Tasa de costos variables)

  • Tasa de costos variables = 1 ' (Ganancia + Costos fijos) / Ingresos

Se nos dan las cantidades de materiales A y B que se utilizan para producir una unidad de cada producto, as como los lmites de uso mensual de los materiales A y B. Adems, se espera que el beneficio por la venta de una unidad del producto X sea de 10,000 yenes y el beneficio por la venta de una unidad del producto Y sea de 20,000 yenes. En este caso, se debe determinar la cantidad de productos X (x unidades) e Y (y unidades) a producir para maximizar el beneficio.

from scipy.optimize import linprog

# Funcin objetivo (dado que se desea maximizar el beneficio, ya que
# scipy.optimize.linprog minimiza, se inverte el signo)
c = [-10, -20]  # -x - 2y

# Condiciones de restriccin
A = [[3, 3],  # 3x + 3y <= 48
     [1, 6]]  # x + 6y <= 36
b = [48, 36]

# Restricciones deno negatividad de las variables
x0_bounds = (0, None)
x1_bounds = (0, None)

# Resolver el problema de programacin lineal
res = linprog(c, A_ub=A, b_ub=b, bounds=[x0_bounds, x1_bounds], method='highs')

# Salida de la solucin ptima
print('Optimal value:', -res.fun, 'with x:', res.x[0], 'and y:', res.x[1])

# Funcin para calcular la cantidad ptima de produccin
x_optimal = res.x[0]
y_optimal = res.x[1]

# x unidades de producto X y y unidades de producto Y
def calculate_profit(x, y):
    return 10 * x + 20 * y

# Calcular el benificio con la cantidad ptima de produccin
profit_optimal = calculate_profit(x_optimal, y_optimal)
print(f"Beneficio con la cantidad ptima de produccin: z = 10 * {x_optimal:.2f} + 20 * {y_optimal:.2f} = {profit_optimal:.2f} ")
