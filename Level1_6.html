Distribucin hipergeomtrica

$ N $ $ M $ $ n $ $ x $

Cuando se realiza un muestreo sin reemplazo de $ n $ elementos de una poblacin finita de $ N $ elementos, en la cual hay $ M $ elementos defectruosos, el nmero de elementos defectuosos $ x $ en la muestra sigue una distribucin hipergeomtrica.

La esperanza y la varianza de una variable aleatria $ x $ que sigue una distribucin hipergeomtrica

$ E(x) = \frac{nM}{N} $

$ V(x) = \frac{nM}{N} (1 - \frac{M}{N}) (\frac{N-n}{N-1}) $

Si $ P = M/N $ se considera la tasa de defectuosos en la poblacin, entonces la esperanza y la varianza mencionadas anteriormente son:

$ E(x) = nP $

$ V(x) = np(1-P)(\frac{N-n}{N-1}) $

Correccin finita (correccin por poblacin finita: $ (\frac{N-n}{N-1}) $

from scipy.stats import hypergeom

# Configuracin de parmetros
N = 50  # Tamao total de la problacin
M = 10  # Nmero de elementos defectuosos
n = 20  # Nmero de muestras a extraer

# distribucin hipergeomtrica
rv = hypergeom(N, M, n)

# Valor esperado y Varianza
expected_value = n * M / N
variance = (n * M / N) * (1 - M / N) * ((N - n) / (N - 1))

#
print(f"Valor esperado (E[X]): {expected_value:.2f}")
print(f"Varianza (Var(X)): {variance:.2f}")

# funcin de masa de probabilidad (PMF)
x = np.arange(0, n + 1)

# Clculo de la funcin de masa de probabilidad (PMF) para cada x

pmf = rv.pmf(x)

# Creacin del grfico
plt.figure(figsize=(10, 6))
plt.bar(x, pmf, color='skyblue', edgecolor='black')
plt.title('distribucin hipergeomtrica (Hypergeometric Distribution)')
plt.xlabel('nmero de artculos defectuosos en la muestra x')
plt.ylabel('funcin de masa de probabilidad (PMF)')
plt.xticks(np.arange(0, n + 1, 1))  #
plt.grid(axis='y', linestyle='--', alpha=0.7)

# Valor esperado y Varianza
plt.axvline(expected_value, color='red', linestyle='dashed', linewidth=1, label=f'Valor esperado (E[X]) = {expected_value:.2f}')
plt.legend()
plt.show()

A $ N=120 $ $ M=12 $ $ x $ $ x $ $ x $ $ \sigma_b^2 $ $ x $ $ \sigma_w^2 $ $ \sigma_b^2 + \sigma_w^2 = \sigma^2 $

Las piezas de los equipos pticos que llegan a la fbrica A se entregan en cajas con $ N=120 $ piezas por caja, y $ M=12 $ cajas por entrega. Dado que la caractersticas $ x $ de las piezas es importante, se realiza un muestreo de las piezas para medir el valor de la caracterstica $ x $ y calcular el valor promedio. Se examimnarn las muestras, considerando la varianza de la carcteristica $ x $ entre las cajas, $ \sigma_b^2 $, la varianza de la caracterstica $ x $ dentro de las piezas de una caja, $ \sigma_w^2 $, y la relacion $ \sigma_b^2 + \sigma_w^2 = \sigma^2 $.

Muestreo en dos etapas

  1. $ M=12 $ $ m=4 $ $ N=120 $ $ n=60 $ $ mn=240 $ $ x $ $ x $ $ \sigma_b^2 = 2^2 $, $ \sigma_w^2 = 1^2 $ $ x $

  2. Se seleccionaron aleatoriamente $ m=4 $ cajas de las $ M=12 $ cajas recibidas, y dentro de cada una de las cajas seleccionadas, se eliqieron aleatoriamente $ n=60 $ piezas de las $ N=120 $ piezas. En total, se midieron las caractersticas $ x $ de $ mn=240 $ piezas. Se debe calcular el valor promedio de la caracterstica $ x $ obtenida. Se debe que $ \sigma_b^2 = 2^2 $, $ \sigma_w^2 = 1^2 $.

# Configuracin de constantes
M = 12  # Nmero de cajas
N = 120  # Nmero de piezas en cada caja
m = 4   # Nmero de cajas seleccionadas
n = 60  # Nmero de piezas seleccionadas de cada caja
sigma_b2 = 4  # Varianza entre cajas
sigma_w2 = 1  # Varianza dentro de cada caja

sample_means = []
for _ in range(1000):
    # Generacin aleatoria del valor medio de cada caja (considerando la varianza enter cajas)
    box_means = np.random.normal(0, np.sqrt(sigma_b2), M)  # Clculo del promedio de cada caja

    # Generacin de la caracteristicas x de las piezas cada caja (considerando la varianza dentro de la caja)
    samples = []
    for i in range(M):
        box_samples = np.random.normal(box_means[i], np.sqrt(sigma_w2), N)  # Aplicacin de la varianza dentro de caja
        samples.append(box_samples)

    # Seleccin aleatoria de m=4
    selected_boxes = np.random.choice(M, m, replace=False)

    # Seleccin de n=60 piezas dentro de las cajas seleccionadas y medicin de la caracteristica x
    selected_samples = []
    for box in selected_boxes:
        selected_samples.extend(np.random.choice(samples[box], n, replace=False))
     # Clculo del valor medio de la caracteristica x
    sample_means.append(np.mean(selected_samples))

print(f"Media de la caracteristica x: {np.mean(sample_means):.2f}")
print(f"Varianza de la caracteristica x: {np.sqrt(np.var(sample_means, ddof=1)):.2f}^2")

# Clculo de la varianza (aplicando la correccin por poblacin final)
var_between = (sigma_b2 / m)*((M-m)/(M-1))  # Varianza entre cajas
var_within = (sigma_w2 / (m * n)) * ((N - n) / (N - 1))  # Varianza total (con correccin por poblacin finita aplicada)

# Varianza
total_variance = var_between + var_within
print(f"Valianza total(con correccin por poblacin finita aplicada): {np.sqrt(total_variance):.2f}^2")

Muestreo estratificado

  1. $ M = 12 $ $ n = 20 $ $ Mn = 240 $ $ x $ $ x $ $ \bar{x} $

  2. Se seleccionaron aleatoriamente $ n=12 $ piezas de cada una de las $ M=20 $ cajas recibidas, obteniendo un total de $ Mn=240 $ mediciones de la caracteristica $ x $. Se calcul el valor medio $ \bar{x} $ de los valores obtenidos. Este tipo de muestreo se denomina muestreo estratificado.

# Configuracin de constantes
M = 12  # Nmero de cajas
N = 120 # Nmero de piezas en cada caja
n = 20  # Nmero de piezas seleccionadas de cada caja
m = 12  # Nmero de cajas seleccionadas

#
sigma_b2 = 4  # Varianza entre cajas
sigma_w2 = 1  # Varianza dentro de cada caja

sample_means = []
for _ in range(1000):
    # Generacin de datos de muestreo estratificado
    mu_global = 0  # Medio total

    # Establecimiento de valores medios diferentes para cada caja
    box_means = np.random.normal(mu_global, np.sqrt(sigma_b2), M)

    # Generacin de datos de las piezas en cada caja
    data = [np.random.normal(mean, np.sqrt(sigma_w2), N) for mean in box_means]

    # Extracin aleatoria de n datos de cada caja
    sampled_data = [np.random.choice(box, n, replace=False) for box in data]

    # Reunin de todas las muestras
    flattened_samples = np.concatenate(sampled_data)

    # Clculo de la media
    x_bar = np.mean(flattened_samples)

    # Clculo de la media de la caracteristica x
    sample_means.append(x_bar)

print(f"Media de la caracterstica x: {np.mean(sample_means):.2f}")
print(f"Varianza de la caracterstica x: {np.sqrt(np.var(sample_means, ddof=1)):.2f}^2")

# Clculo de la varianza (aplicando la correccin por poblacin finita)
m = M
var_between = (sigma_b2 / m)*((M-m)/(M-1))  # Varianza entre cajas
var_within = (sigma_w2 / (m * n)) * ((N - n) / (N - 1))  # Varianza dentro de las cajas (correccin por poblacin finita)

# Varianza total
total_variance = var_between + var_within
print(f"Varianza total (con correccin por poblacin finita aplicada): {np.sqrt(total_variance):.3f}^2")

#
plt.hist(sample_means, bins=10, edgecolor='black', alpha=0.7)
plt.xlabel('Caracteristica x')
plt.ylabel('Frecuencia')
plt.title('Distribucin de la caracteristica x mediante muestreo estaratificado')
plt.grid()
plt.show()

Varianza de la caracterstica x: 0.58^2
Varianza total (con correccin por poblacin finita aplicada): 0.059^2

La principal ventaja del muestreo estratificado es minimizar el impacto de la variabilidad entre las cajas, lo que reduce la dispercin total del la muestra. Esto permite mejorar la precisin incluso con un menor tamao de muestra.
