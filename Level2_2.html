import numpy as np

# データの生成
np.random.seed(0)  # 再現性のためにシードを設定

# 魅力的品質のデータ
physical_fulfillment_attractive = np.random.rand(100) * 100
customer_satisfaction_attractive = adjust_response(physical_fulfillment_attractive)/2 + 50 +  np.random.randn(100) * 10


# 一元的品質のデータ
physical_fulfillment_one_dimensional = np.random.rand(100) * 100
customer_satisfaction_one_dimensional = physical_fulfillment_one_dimensional + np.random.randn(100) * 10

# 当たり前品質のデータ
physical_fulfillment_must_be = np.random.rand(100) * 100
customer_satisfaction_must_be = np.clip(physical_fulfillment_must_be / 2 + np.random.randn(100) * 10, 0, 50)

# データをクリップして、0-100の範囲に収める
customer_satisfaction_attractive = np.clip(customer_satisfaction_attractive, 0, 100)
customer_satisfaction_one_dimensional = np.clip(customer_satisfaction_one_dimensional, 0, 100)
customer_satisfaction_must_be = np.clip(customer_satisfaction_must_be, 0, 100)

魅力的品質（Attractive Quality）: 物理的充足状況が低くても顧客満足度が高いことが多い。

一元的品質（One-Dimensional Quality）: 物理的充足状況が高いほど顧客満足度も高くなる。

当たり前品質（Must-Be Quality）: 物理的充足状況が低いと顧客満足度が非常に低くなるが、充足状況が高くなると満足度がある程度までしか上がらない。
