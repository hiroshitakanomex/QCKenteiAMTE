Diagrama de control

05-01 Tipos de diagramas de control

1. Qu es un diagrama de control

El diagrama de control es una herramienta creada por W.A. Shewahatt para analizar u controlar la variabilidad de un precoso.

  • Causas aleatorias: Variablidad inevitable
  • Causas especiales: Variabilidad que no se puede pasar por alto

2. Tipos de diagramas de control

  • Diagrama de control de la media y el rango: Diagrama de control $ \bar{X}-R $
  • Diagrama de control del la mediana y e rango: Diagrama de control $ Me-R $
  • Diagrama de control de la media y el error estndar: Diagrama de control $ \bar{X}-s $
  • Diagrama de control de los datos individuales y el rango mvil: Diagrama de control $ X - Rm $
  • Diagrama de control de la tasa de defectos: Diagrama de control $ p $
  • Diagrama de control del nmero de defectos: Diagrama e control $ np $
  • Diagrama de control del nmero de defectos por unidad: Diagrama de control $ u $
  • Diagrama de control del nmero de defectos: Diagrama de control $ c $

05-02 Cmo eleborar un diagrama de control

1. Como eleborar un diagrma de contro $ \bar{X} - R $

Paso 1: Recoger los datos y agruparlos

Paso 2: Calcular la mdedia X y el rango R de cada grupo

Paso 3: Calcular la media total X y el promedio del rango R

Paso 4 Determinar las lneas de control

2. Diagrama de control de la mediana y el rango (Diagrama $ Me-R $)

  1. Se puede reducir el impacto de los valores atpicos dentro del grupo.
  2. So el tamao del grupo es impar, no es necesario calcular Me, lo que lo hace ms sencillo.
  3. Es comparacin con el diagrama de control $ \bar{X} $, la tasa de deteccin es peor.

05-04 ndice de capacidad del proceso

1. Qu es el ndice de capacidad del proceso

El ndice de capacidad del proceso es un indicador que mide la capacidad cualitativa del proceso para producir productos. Se utilizan los ndices de capacidad del proceso $ C_p $ y $ C_{pk} $, que se calculan a partir de la relacin entre la variabilidad del proceso y el rango de las especificaciones del producto.

2. Cmo calcular el ndice de capacidad del proceso

ndice de capacidad del proceso para especificaciones bilaterales

$ C_p = \frac{S_u - S_L}{6 \sigma} $

En el caso de solo lmite superior de especificacin:

$ C_p = \frac{S_U - \bar{X}}{3\sigma}$

En el caso de solo lmite inferior de especificacin:

$ C_p = \frac{\bar{X}-S_L}{3\sigma} $

ndice de capacidad del proceso considerando el sesgo:

El menor de los dos

$ C_p = \frac{S_U - \bar{X}}{3\sigma}$, $ C_p = \frac{\bar{X}-S_L}{3\sigma} $

3. La evaluacin del ndice de capacidad del proceso

Generalmente, si $ C_p \gt 1.33 $, se considera que la capacidad del proceso es suficiente.

import scipy.stats as stats
import math

# Limites superior e inferior de especificacin
Su = 100  # Lmite superior de especificacin
SL = 40   # Lmite inferior de especificacin

# Valor promedio y desviacin estndar de los datos
Xbar = 70  # Valor promedio
s = 7.5    # Desviacin estndar
n = 10     # Nmer de datos

# Clculo de Cp
Cp = (Su - SL) / (6 * s)

# Clculo del intervalo de confianza del 95% para la varianza
alpha = 0.05
chi2_lower = stats.chi2.ppf(1 - alpha / 2, df=n - 1)  # Valor superior de 
chi2_upper = stats.chi2.ppf(alpha / 2, df=n - 1)      # Valor crtico inferior de 

# Intervalo de confianza para la desviacin estndar
# (calculado a partir de intervalo de confianza para la varianza)
variance_lower = ((n - 1) * s**2) / chi2_lower
variance_upper = ((n - 1) * s**2) / chi2_upper
stddev_lower = variance_lower**0.5
stddev_upper = variance_upper**0.5

# Clculo del intervalo de confianza para Cp
Cp_lower = (Su - SL) / (6 * stddev_upper)
Cp_upper = (Su - SL) / (6 * stddev_lower)

# Visualizacin de los resultados
print(f"Cp: {Cp:.2f}")
print(f"Intervalo de confianza del 95%: ({Cp_lower:.2f}, {Cp_upper:.2f})")

import scipy.stats as stats

# Lmite inferior de especificacin
SL = 40

# Promediodesviacin estndartamao de la muestra de los datos
Xbar = 70  # Promedio
s = 7.5    # Desviacin estndar
n = 10     # Tamao de la muestra

# Clculo de Cpen el caso de solo el lmite inferior de especificacion
Cp = (Xbar - SL) / (3 * s)

# Valor z
z = stats.norm.ppf(0.95)

# Clculo de la varianza de Cp considerando la varianza de Cp en s
Cp_variance = (Cp ** 2) / (2*(n - 1)) + (1 / (9 * n))

# Clculo del ancho del intervalo de confianza de Cp
Cp_conf_interval_width = z * np.sqrt(Cp_variance)

# Intervalo de confianza de Cp
Cp_lower = Cp - Cp_conf_interval_width
Cp_upper = Cp + Cp_conf_interval_width

# Mostrar resultados
print(f"Cp: {Cp:.2f}")
print(f"Intervalo de confianza de 95%: ({Cp_lower:.2f}, {Cp_upper:.2f})")
