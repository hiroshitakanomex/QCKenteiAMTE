06-01 Inspeccin por muestreo

La inspeccin por muestreo es el proceso mediante el cual se toma una muestra de un lote objetivo siguiendo un mtodo de inspeccin por muestreo previamente establecido. Luego, se realizan pruebas o mediciones en la muestra y se comparan con los ciriterios de aceptacin o rechazo del lote, para determinar si dicho lote es aceptado o rechazado.

1. Qu es la inspeccin por muestreo?

  1. Clasificacin segn los datos menejados
  • Inspeccin por muestreo de datos por conteo
  • Inspeccin por muestreo de datos por medicin
  1. Clasificacin segn el nmero de mestreos
  2. Clasificacin segn el mtodo para determinar los criterios de aceptacin o rechazo

2. Tipos de mtodos de inspeccin por muestreo

3. Diseo previo de la inspeccin por muestreo

  • Curva probabilidad acumlada (Culva Thorndike-Haga)

# Rango de tasa de defectos (p)

06-02 Inspeccin por muestreo de tipo de conteo

La inspeccin por muestreo de tipo de conteo acepta el lote aprobado tal como est, pero en el caso del lote no aprobado, se realiza una seleccin total para reparar los productos defectuosos o reemplazarlos por productos conformes, asegurando que todo el lote sea conforme.

  • AOQ: Avergae Outgoing Quality
  • LTPD: Lot Tolerance Percent Defective
  • AOQL: Average Outgoing Quality Limit
  • Curva de Calidad Promedio de Aceptacin
  • AOQL: Lmite de Calidad Promedio de Salida

06-03 Inspeccin por muestreo tipo ajustado

Inspeccin por muestreo tipo ajustado

La inspeccin por muestreo tipo ajustado se caracterza por disponer de tres tipos de tablas de muestreo: normal, estricat y relajada, y cambiar el nivel de aplicacin segn el estado de los resultados de la inspeccin.

Inspeccin por muestreo segn JIS Z 9015-1:2006

Una normativa que regula el mtodo de inspeccin por muestre obasado en el ndice AQL (Nivel de Calidad Aceptable) para la inspeccin por lotes.

Caractersticas de JIS Z 9015-1:2006

Procedimiento de inspeccin

  1. Establecer los criterios de evaluacin de calidad para clasificar productos conformes y no conformes.
  2. Determinar el AQL (Nivel de Calidad Aceptable)
  3. Establecer el nivel de inspeccin
  4. Determinar el tipo de muestreo
  5. Determinar el grado de rigor de la inspeccin.
  6. Tormar los lotes.
  7. Determinar el mtodo de muestreo.
  8. Realizar la inspeccin de muestreo sobre las muestras.
  9. Getional los lotes aprobados y rechazados.
  10. Procesar los lotes.
  11. Registrar los resultados de la inspeccinl incluyendo los elementos necesarios para ajustar el rigor de la inspeccin.

from scipy.optimize import brentq

def get_sample_code_letter(lot_size, inspection_level):
    # Obtener el carcter de cdigo de muestra basado en la tabla
    inspection_levels = {
        'I': [2, 8, 15, 25, 50, 90, 150, 280, 500, 1200, 3200, 10000, 35000, 150000, 500000],
        'II': [2, 8, 20, 32, 50, 125, 200, 315, 500, 1250, 3200, 10000, 35000, 150000, 500000],
        'III': [2, 8, 20, 32, 80, 125, 315, 500, 800, 2000, 3200, 10000, 35000, 150000, 500000]
    }

    lot_ranges = [2, 8, 15, 25, 50, 90, 150, 280, 500, 1200, 3200, 10000, 35000, 150000, 500000]
    code_letters = ['A', 'B', 'C', 'D', 'E', 'F', 'G', 'H', 'J', 'K', 'L', 'M', 'N', 'P', 'Q']

    for i, size in enumerate(lot_ranges):
        if lot_size <= size:
            return code_letters[i]
    return code_letters[-1]

def get_sampling_plan(code_letter, aql):
    # Obtener el tamao de la muestra y los criterios de aceptacin basados en la tabla
    sampling_table = {
        'A': (2, 0, 1),
        'B': (2, 0, 1),
        'C': (2, 0, 1),
        'D': (3, 0, 1),
        'E': (5, 0, 1),
        'F': (8, 1, 2),
        'G': (13, 1, 2),
        'H': (20, 1, 2),
        'J': (32, 2, 3),
        'K': (50, 3, 4),
        'L': (80, 4, 5),
        'M': (125, 6, 7),
        'N': (200, 8, 9),
        'P': (315, 10, 11),
        'Q': (500, 10, 11)
    }
    return sampling_table[code_letter]


# main
lot_size = 1000
inspection_level = 'III'
aql = 0.015

# Obtener el carcter de cdigo
code_letter = get_sample_code_letter(lot_size, inspection_level)
print(f"Sample Code Letter: {code_letter}")

# Obtener el tamao de la muestra y los criterios de aceptacin
n, ac, re = get_sampling_plan(code_letter, aql)

p_values = np.linspace(0, 0.2, 1000)

# 1. La tasa de aceptacin cuando la tasa de no conformidad del lote es del 1.5%
p_defect = 0.015  # Tasa de no conformidad 1.5%
accept_prob = sum(binom.pmf(k, n, p_defect) for k in range(ac + 1))
print(f"La probabilidad de aceptacin cuando la tasa de no conformidad del lote es del 1.5%: {accept_prob:.4f}")

# 2. Tasa de riesgo del productor (la tasa de rechazo cuando la tasa de no conformidad es baja)
# La tasa de rechazo cuando la tasa de no conformidad es extremadamente baja.
producer_risk = 1 - sum(binom.pmf(k, n, 0.01) for k in range(re + 1))
print(f"Tasa de riesgo del productorla probabilidad de rechazo cuando la tasa de no conformidad es del 1%: {producer_risk:.4f}")

# 3. Determinar la tasa de no conformidad cuando la tasa de aceptacin es del 10%
def find_defect_rate(p):
    return sum(binom.pmf(k, n, p) for k in range(ac + 1)) - 0.1

# Encontrar la raz utilizando el mtodo de Brent
p_accept_10 = brentq(find_defect_rate, 0.001, 0.5)
print(f"La tasa de no conformidad del lote cuando la probabilidad de aceptacin es del 10%: {p_accept_10 * 100:.2f}%")
